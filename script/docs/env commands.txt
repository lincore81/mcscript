Legend:
	x|y			either x or y
	<dir>		an argument
	(x)			optional parameter
	{x}			once or more

/env info engines
/env info engine <engine>
/env info threads
/env info $<file>

/env alias list|ls
/env alias set <name> <value>
/env alias remove <name>

/env files|fs manager|mgr (<dir>)
/env files|fs list|ls (<dir>)
/env files|fs cat <file>

/env config|cfg list|ls
/env config|cfg set <key1> = <value1> {, <keyN> = <valueN>}
/env config|cfg get <key>
/env config|cfg remove <key>
				
/env help (<topic>)
